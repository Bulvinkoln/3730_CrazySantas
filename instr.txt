Работа с ГИТ

Для начала работы с ГИТ необходимо представиться

Для этого вводятся команды:

$ git config --global user.name "имя"

$ git config --global user.email "почта"

Чтобы начать работу  на своем компьтере в общем проекте, необходимо:

   1. "форкнуть" репозиторий на свой аккаунт

   2. кллнировать его по ссылке на компьютер

   3. открыть клоп=нированную папку

   4. создать новую ветку для внесения изменений, чтобы чистовой проект не изменялся

   5. внести свои изменения

   6. "запушить" изменения в свой удаленный репозиторий

   7. сделать pull request
