### Работаю с ветками

### Чтобы создать ветку, используется команда *git branch <название новой ветки>*

### Чтобы сменить ветку, надо использовать команду *git checkout <название ветки>*

### Чтобы слить ветки, надо использовать команду *git merge <название ветки>*

### Чтобы удалить ветку, надо использовать команду *git branch -d <название ветки>*
